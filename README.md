# Care n Cure Acupuncture Clinic Website

This is a premium, mobile-first website for Care n Cure Acupuncture Clinic, designed to showcase treatments, patient testimonials, and facilitate appointment bookings.

## Features
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **Modern Aesthetics**: Spa-like, medical, and premium look.
- **Sections**: Home, About, Services, Appointment Booking, Testimonials, Gallery, Blog, Contact.
- **Integration**: WhatsApp chat and appointment booking buttons.

## Tech Stack
- HTML5
- CSS3 (Vanilla, Custom Properties)
- JavaScript (Vanilla)
- Vite (Build Tool)

## Project Setup

### Prerequisites
- Node.js (v18 or higher recommended)

### Installation
Run the following command to install dependencies:
```bash
npm install
```

### Development
To start the local development server:
```bash
npm run dev
```
Then open http://localhost:5173 in your browser.

### Build
To build the project for production:
```bash
npm run build
```
The output will be in the `dist` folder.

## Deployment
This project is ready for deployment on platforms like **Vercel**, **Netlify**, or **GitHub Pages**.

### Deploy to Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project root.
3. Follow the prompts.

For manual deployment, upload the contents of the `dist` folder to your hosting provider.
